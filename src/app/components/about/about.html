El proyecto PawFeeder

PawFeeder nace como un proyecto universitario de IoT con el objetivo de mejorar el bienestar animal mediante tecnología accesible y de código abierto.

¿Qué es?

Un dispensador inteligente para mascotas

PawFeeder combina hardware Arduino con una aplicación web/móvil para ofrecer un sistema completo de alimentación automatizada. El dispositivo se conecta a tu red WiFi y responde a comandos en tiempo real.

Puedes programar horarios fijos, activar comidas manualmente desde cualquier lugar y recibir notificaciones cuando el nivel de alimento sea bajo.

  • Control remoto via WiFi / MQTT
  • Sensor ultrasónico para medir nivel de croqueta
  • App web responsiva con prototipo móvil
  • Historial de comidas y reportes
